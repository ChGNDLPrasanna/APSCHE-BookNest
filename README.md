# APSCHE BookNest

## Project Overview
BookNest is a full-stack online book shopping platform developed using the MERN Stack (MongoDB, Express.js, React.js, Node.js).

### Features
- User Authentication
- Book Search and Filtering
- Shopping Cart
- Wishlist
- Reviews and Ratings
- Seller Dashboard
- Order Management
- Payment Gateway Integration

### Technologies Used
- React.js
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- Tailwind CSS

### Team Project
This project was developed as part of an APSCHE internship/team project.
